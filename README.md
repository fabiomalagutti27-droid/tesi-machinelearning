# tesi-machinelearning
Machine Learning per la stima della temperatura superficiale urbana da indici satellitari: applicazione a Bologna
- in data.zip sono presenti tutti i dati che sono stati utilizzati nei vari run del notebook. 
- nel notebook sono presenti tutti i blocchi di codice da runnare in modo sequenziale e blocchi di testo che ne descrivono l'utilizzo.

Per runnare: 
- caricare il notebook su google colab
- creare una cartella chiamata "data" nella directory corrente di google colab e fare l'upload di tutti i file contenuti in data.zip
- runnare sequenzialmente le celle di codice
